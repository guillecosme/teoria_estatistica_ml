## Como usar

`conceitos_detalhados.md` é o coração do caderno: 1.985 linhas de referência. Não leia linear — use como consulta. Cada seção é autossuficiente.

`referencias_estudo.md` complementa: quando o conceito ainda não estiver claro depois de ler aqui, vá para os recursos externos.
